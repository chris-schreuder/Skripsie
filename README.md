# Skripsie Use Instructions
There are two folders, one for the completed per simulation game (PetSim), and one for all the model experiments (notebook)

# PetSim
To run this project, you will need:
- Python 3.10.5
- install all requirements by running "pip install -r requirements.txt" in the command line in the PetSim folder directory
- run the main.py file
- go to the generated link to few web-application

# notebook
All the experements can be found in the "Experiment" folder as:
- Basic DTW experiment : DTW_Exp.ipynb
- Minimum QBE DTW experiment : QBE_Exp.ipynb
- Average QBE DTW experiment : QBE_Exp_Avg.ipynb
- Segmental DTW experiment : Seg_Exp.ipynb
- General functions : DTW_functions.ipynb
- QBE segments alignment/scaling : QBE_scaling.ipynb
- User review visuals : User_Review_Graph.ipynb
